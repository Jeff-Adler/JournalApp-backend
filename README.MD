# JournalApp

This is just a basic journal app to get back on the swing of web dev.

- Backend Stack: Python, FastApi, SQLAlchemy

- Frontend Stack: TypeScript, React

Routes:

# - GET user
# - GET user by email
# - GET all user
# - POST user
# - GET particular task by id
# - GET a user's tasks
# - POST new task
# - PATCH task: task
# - PATCH task: mark as done
# - DELETE task by id

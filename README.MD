# JournalApp

This is just a basic journal app to get back on the swing of web dev.

- Backend Stack: Python, FastApi, SQLAlchemy

- Frontend Stack: TypeScript, React

Routes:

- GET user
- GET user by email
- GET all user
- POST user
- GET all notes
- GET particular note by id
- GET a user's notes
- POST new note
- PATCH note: task
- PATCH note: mark as done
- DELETE note by id
